Começando no HTML

Um resumo para revisar o básico do html

O que é o HTML?

O HTML é uma linguagem de marcação baseada em hipertexto.

Tags de formatação de texto

Tag mark

Destaca o texto como se estivesse marcado com um marca-texto. Exemplo: texto destacado.

Tag small

Exibe o texto menor do que o tamanho padrão. Exemplo: texto menor.

Tag i

Deixa o texto em itálico (ênfase semântica leve ou estilo). Exemplo: texto em itálico.

Tag u

Sublinha o texto (geralmente usado para indicar erro ou destacar algo). Exemplo: texto sublinhado.

Tag strong

Indica um texto com forte importância (fica em negrito). Exemplo: texto importante.

Listas

Tags ol, ul e li

A tag ol cria uma lista ordenada (com números).

A tag ul cria uma lista não ordenada (com marcadores).

A tag li define um item dentro de uma lista. Exemplos:

    Lista ordenada
  1. item da lista
    Lista não ordenada
  • item da lista

Subscrito e sobrescrito

Tag sub

Deixa o texto em subscrito (usado para fórmulas químicas, por exemplo). Exemplos: texto subscrito; H2SO4 (ácido sulfúrico).

Tag sup

Deixa o texto em sobrescrito (usado para expoentes). Exemplos: texto sobrescrito; 211= 2048.

Citações blockquote

Indica um bloco de citação, geralmente exibido com recuo. Exemplo:

"Minha terra tem palmeiras Onde canta o Sabiá." – Gonçalves Dias
