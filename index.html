O que é o HTML?

O HTML é uma linguagem de marcação baseada em hipertexto.

Tags de formatação de texto

Tag mark

Destaca o texto como se estivesse marcado com um marca-texto. Exemplo: texto destacado.

Tag small

Exibe o texto menor do que o tamanho padrão. Exemplo: texto menor.

Tag i

Deixa o texto em itálico (ênfase semântica leve ou estilo). Exemplo: texto em itálico.

Tag u

Sublinha o texto (geralmente usado para indicar erro ou destacar algo). Exemplo: texto sublinhado.

Tag strong

Indica um texto com forte importância (fica em negrito). Exemplo: texto importante.
